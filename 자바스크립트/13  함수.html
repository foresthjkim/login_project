매개변수가 있는 함수 정의문

매개변수가 있는 함수 정의문은 함수를 호출할때 전달 값을 입력하여 호출할 수 있다. 이렇게 전달된 값은 매개변수가 받아 정의문 내에 일련의 실행문에 사용할 수 있게 된다.

데이터와 매개변수는 선언된 순서대로 매칭된다. 즉, 첫번쨰 매개변수는 첫번쨰 데이터의 값을 전달 받는다.

데이터를 반환하는 return 문 - 값을 반환하고 종료한다.

return 강제종료 예: return문 뒤에 나오는 실행문은 실행이 되지 않는다.

재귀함수 호출 - 재귀함수를 호출하여 1부터 10까지 값을 출력
