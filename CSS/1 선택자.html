타입 선택자

가장 기본적인 선택자는 HTML요소 이름을 사용하는 것이다. 예를 들어 <P> 요소를 선택하려면 P라고 적어주면 된다. 이것을 타입 선택자(type selector)이라고 한다.

아이디 선택자

  • 특정한 요소를 선택하고자 할 때 사용한다
  • html요소에 id를 부여한다.
  • CSS에서 요소의 id앞에 #을 붙이면 된다.
  • 현재 문서에 id는 하나이어야 한다. 동일한 id가 존재하면 안된다.
  • 아이디명은 숫자로 시작하면 안된다.

클래스 선택자

  • 몇 개의 요소를 하나의 클래스로 묶어서 스타일을 지정하고자 할때 사용한다.
  • html요소에 클래스를 부여한다.
  • CSS에서 클래스명 앞에 .을 붙이면 된다.
  • 클래스 선택자는 여러개의 요소를 한꺼번에 선택할 수 있다.
  • 클래스명은 숫자로 시작하지 않는다.

class선택자는 class로 붙인 이름을 가져다 쓰는 선택자입니다. '.'을 붙여 표기합니다.

전체 선택자

전체 선택자를 *기호로 표시되며 페이지 안 모든 요소를 선택할 때 사용한다.

자식 선택자

  • 현재 요소의 바로 아래에 나오는 선택자
  • ul>li라고 표현하면 ul에 포하묀 li요소를 말한다.

ul과 p는 형제관계입니다.

인접 선택자

현재 요소 밖의 바로 뒤에 나오는 요소를 가리키는 선택자 입니다.

인접 선택자는 +를 붙입니다.
