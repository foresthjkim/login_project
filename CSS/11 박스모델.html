패딩 : 박스의 안쪽 여백을 말합니다.

박스의 안쪽 여백을 말합니다.

  • length : px, pt, cm 단위로 지정할 수 있다. 음수값은 사용할 수 없습니다.
  • % :상위 요소의 크기를 참조하여 비율로 결정합니다.
  • padding:10px
  • padding:10px 20px : 상하10px, 좌우20px
  • padding:10px 20px 30px : 상 좌우 하
  • padding:10px 20px 30px 40px : 상 우 하 좌 (시계방향)

margin : 박스의 바깥쪽 여백 (border 바깥쪽 여백을 지정)

  • length : px, pt, cm 단위로 지정할 수 있다. 음수값은 사용할 수 없습니다.
  • % :상위 요소의 크기를 참조하여 비율로 결정합니다.
  • margin: 10px
  • margin:10px 20px : 상하10px, 좌우20px
  • margine:10px 20px 30px : 상 좌우 하
  • margine:10px 20px 30px 40px : 상 우 하 좌 (시계방향)
