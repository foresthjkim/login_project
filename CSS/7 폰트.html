색상

컴퓨터에서 색상은 빛의 3원색인 빨강색(red), 녹생(green), 청색(blue)를 혼합하여 만들어진다. CSS에서 색상을 표현하는 몇 가지 방법이 있다.

배경색은 노랑, 글자색은 빨간색으로 표현해 보세요.

  • 이름으로 표현하기
  • 16진수로 표현하기
  • 10진수로 표현하기
  • 배경색 opacity 50% 적용하기
  • 퍼센트로 표현하기

서체 - 폰트의 종류

폰트이 종류를 설정할 수 있습니다. font-family 속성으로 설정할때는 여러개의 폰트 이름을 제공하는 것이 좋다. 왜냐하면 브라우저가 HTML파일을 표시할 때 클라이언트 컴퓨터에 지정된 폰트가 없는 경우도 있기 때문이다. 첫번째 폰트가 없으면 자동적으로 다음 폰트를 시도한다.

폰트를 나열할 때는 첫 번째 폰트는 개발자가 가장 원하는 폰트, 맨 마지막에는 가장 일반적인 폰트를 지정한다.

만약 폰트 이름이 여러 단어로 되어 있다면 큰따옴표를 사용하여 묶어야 한다.

웹 폰트

웹 개발자는 사용자의 컴퓨터를 제어할 수 없다. 따라서 사용자의 컴퓨터에 폰트를 설치할 수 없다. 그러면 어떻게 하면 좋을까?

이런 경우 웹폰트(web font)를 사용할 수 있다.

웹 폰트는 폰트를 웹서버에게 저장해 두었다가 필요할때 사용자의 웹브라우저로 직접 전송하는 기법이다.
English Font test: Web font can be imported/stored in the Web Server, and it is being directly transmitted to web browser of users.

폰트 사이즈

font-size이며 px, %, em단위를 사용할 수 있습니다. w3c에서는 em단위를 권장합니다. em은 배수를 말합니다. 폰트는 지정하지 않으면 대략 16px정도의 크기를 나타냅니다.
