border : 박스의 테두리

  • border-style : 박스 테두리의 선 스타일을 지정하는 속성
  • border-color : 선 색상을 지정하는 속성
  • border-width : 테두리 선의 굵기
  • border : 경계선 스타일을 한 줄에 한꺼번에 설정할 수 있다. border-width, border-style(필수), border-color순서로 적어 주어야 한다.

border-radius : 둥근 모서리

  • border-radius : 등근 코너의 반지름을 나타냅니다.
  • 각 방향별로 값을 지정할 수 있으며 top-left, top-right, bottom-right, bottom-left순입니다.
  • width값과 height값의 1/2을 지정하면 동그란 원을 표현할 수 있다. (패딩값 포함)

box-shadow : 박스 그림자

box-shadow : 가로오프셋(x), 세로오프셋(y), 번지는 정도, 그림자색;

inset을 써주면 박스의 그림자를 안쪽으로 퍼지게 할 수 있다.

가로세로 오프셋은 주지 안흔 것이 좋아요. box-shadow : 가로오프셋 세로오프셋 번지는 정도 그림자와박스와의거리 그림자색;

박스의 크기 게산

box-sizing : 박스의 크기를 결정하는 방식

content-box : 요소 박스에 지정된 모든 값을 더해 박스의 크기를 결정하는 방식 (기본값)

border-box : 콘텐츠에 지정된 너비에 안쪽 여백과 테두리를 포함 시키는 방식

width : 200px; padding : 10px; border : 5px solid red; margin : 20px; 전체 박스 크기는? (270=20+5+10+200+10+5+20)
width : 200px; padding : 10px; border : 5px solid red; margin : 20px; box-sizing:border-box;전체 박스 크기는? (240=20+200+20)
