# 요구사항 정의서

## 프로젝트 목적

건강검진 데이터를 기반으로 사용자의 비만 여부를 예측하는 AI 웹 서비스를 개발한다.

Spring Boot 서버와 Flask AI 서버를 분리하여 REST API 기반 서버 연동 구조를 학습하고,
머신러닝 모델을 웹 서비스에 적용하는 경험을 목표로 한다.

---

## 사용자 요구사항

### 비회원 사용자

- 메인 화면에서 키와 몸무게 데이터를 입력할 수 있다.
- 입력한 데이터를 기반으로 AI 예측 결과를 확인할 수 있다.
- 회원가입 없이 비만 여부 예측 기능을 사용할 수 있다.

### 회원 사용자

- 회원가입을 할 수 있다.
- 로그인을 할 수 있다.
- 로그아웃을 할 수 있다.
- 게시글 목록을 조회할 수 있다.
- 게시글 상세 내용을 조회할 수 있다.
- 게시글을 작성할 수 있다.
- 본인이 작성한 게시글을 수정할 수 있다.
- 본인이 작성한 게시글을 삭제할 수 있다.
- 비만 여부 예측 기능을 사용할 수 있다.

---

## 시스템 요구사항

### Spring Boot 서버

- 사용자 요청을 처리한다.
- Thymeleaf 기반 화면을 렌더링한다.
- 회원가입 및 로그인 요청을 처리한다.
- Spring Security 기반 인증을 처리한다.
- 게시판 CRUD 기능을 처리한다.
- 사용자가 입력한 예측 데이터를 Flask AI 서버로 전달한다.
- Flask AI 서버에서 반환한 예측 결과를 화면에 출력한다.
- JPA를 사용하여 MySQL 데이터베이스와 연동한다.

### Flask AI 서버

- 학습된 머신러닝 모델을 로드한다.
- Spring Boot 서버에서 전달받은 입력 데이터를 검증한다.
- 입력 데이터를 머신러닝 모델에 전달하여 비만 여부를 예측한다.
- 예측 결과를 JSON 형식으로 반환한다.
- 서버 상태 확인용 API를 제공한다.

### MySQL 데이터베이스

- 회원 정보를 저장한다.
- 게시글 정보를 저장한다.
- 필요한 경우 예측 결과 데이터를 저장할 수 있다.

## 기능 요구사항

### 회원 기능

- 사용자는 아이디, 비밀번호, 이름 등의 정보를 입력하여 회원가입할 수 있다.
- 사용자는 가입한 계정으로 로그인할 수 있다.
- 비밀번호는 암호화하여 저장한다.
- 로그인한 사용자는 로그아웃할 수 있다.

### 게시판 기능

- 사용자는 게시글 목록을 조회할 수 있다.
- 사용자는 게시글 상세 페이지를 조회할 수 있다.
- 로그인한 사용자는 게시글을 작성할 수 있다.
- 게시글 작성자는 본인의 게시글을 수정할 수 있다.
- 게시글 작성자는 본인의 게시글을 삭제할 수 있다.

### 예측 기능

입력값:

- 키
- 몸무게

출력값:

- 비만 여부 예측 결과

처리 흐름:

1. 사용자가 웹 화면에서 키와 몸무게를 입력한다.
2. Spring Boot 서버가 입력 데이터를 받는다.
3. Spring Boot 서버가 Flask AI 서버의 예측 API를 호출한다.
4. Flask AI 서버가 머신러닝 모델로 예측을 수행한다.
5. Flask AI 서버가 예측 결과를 JSON으로 반환한다.
6. Spring Boot 서버가 예측 결과를 화면에 출력한다.

## API 요구사항

### Spring Boot 화면 요청

| 기능 | Method | URL | 설명 |
| --- | --- | --- | --- |
| 메인 화면 | GET | `/` | 메인 화면 조회 |
| 회원가입 화면 | GET | `/signup` | 회원가입 화면 조회 |
| 회원가입 처리 | POST | `/signup` | 회원가입 요청 처리 |
| 로그인 화면 | GET | `/login` | 로그인 화면 조회 |
| 로그아웃 | POST | `/logout` | 로그아웃 처리 |
| 게시글 목록 | GET | `/question/list` | 게시글 목록 조회 |
| 게시글 상세 | GET | `/question/detail/{id}` | 게시글 상세 조회 |
| 게시글 작성 화면 | GET | `/question/create` | 게시글 작성 화면 조회 |
| 게시글 작성 처리 | POST | `/question/create` | 게시글 작성 요청 처리 |
| 예측 요청 | POST | `/predict` | 비만 여부 예측 요청 |

### Flask AI 서버 API

| 기능 | Method | URL | 설명 |
| --- | --- | --- | --- |
| 서버 상태 확인 | GET | `/health` | Flask 서버 상태 확인 |
| 비만 예측 | POST | `/predict` | 입력 데이터 기반 비만 여부 예측 |

## 비기능 요구사항

- 반응형 UI를 지원한다.
- Bootstrap을 활용하여 화면을 구성한다.
- Spring Boot와 Flask 서버는 REST API 방식으로 통신한다.
- 프로젝트 소스 코드는 GitHub로 형상 관리한다.
- 주요 기능과 실행 방법을 README 문서에 정리한다.
- 예측 실패나 서버 오류 발생 시 사용자에게 안내 메시지를 제공한다.
