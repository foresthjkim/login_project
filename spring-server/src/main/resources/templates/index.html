Spring Boot + Flask AI

건강검진 데이터 기반 비만 예측 서비스

키와 몸무게를 입력하면 Flask AI 서버가 비만 여부를 예측하고, Spring Boot 화면에서 결과를 확인할 수 있습니다.

AI 예측 시작 게시판 보기

서비스 흐름

  1. 사용자가 건강 정보를 입력합니다.
  2. Spring Boot가 Flask API를 호출합니다.
  3. AI 예측 결과를 화면에 출력합니다.

주요 기능

포트폴리오에서 설명하기 좋은 핵심 기능을 분리했습니다.

회원 / 로그인

Spring Security 기반 회원가입, 로그인, 로그아웃 기능을 제공합니다.

게시판

로그인 사용자가 게시글을 작성하고 본인 글을 수정하거나 삭제할 수 있습니다.

AI 비만 예측

비회원도 키와 몸무게를 입력해 AI 예측 결과를 확인할 수 있습니다.

로그인 상태입니다. 게시판 작성과 AI 예측 기능을 사용할 수 있습니다.
